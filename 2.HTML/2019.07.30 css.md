# CSS

1. Inline

```html

```
2. Embedding
```html
<head>
    <style>
        h2{
            color:blue;
			font-size:100px;
        }
    </style>
</head>
```

3. Link file(외부참조)

html과 같은 파일명.css 만들기

```html
<head>
	<link rel="stylesheet" href="00_intro.css"
</head>
```
